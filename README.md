# GitPython
GPS watch for Raspberry Pi Pico by MicroPython<br>
1) Use ST7789 IPS display 240x240 SPI<br>
2) Use GY-GPS6MV2 for real time clock<br>

#SPI(1) default pins<br>
spi1_sck   = 10<br>
spi1_mosi  = 11<br>
spi1_miso  =  8     #not use<br>
st7789_res = 12<br>
st7789_dc  = 13<br>
uart-tx    = 0<br>
uart-rx    = 1<br>
